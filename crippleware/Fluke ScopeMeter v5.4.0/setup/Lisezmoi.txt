   1. VERSION DEMO, ACTIVATION ET MISES À JOUR
      Cette version de FlukeView est une version de démonstration
      avec des fonctionnalités limitées. Pour obtenir la version 
      complète, un code d'activation fournit lors de l'achat de 
      FlukeView devra être saisi dans la fenêtre contextuelle 
      d’Activation.
      Pour utiliser FlukeView avec des fonctionnalités limitées, 
      cliquez sur DEMO dans la fenêtre Activation.
      Cette version permet également aux utilisateurs de la version 
      FlukeView V4.x de la mettre à jour vers une version 
      complète V5.x
   ________________________________________________________________

   3. LECTURE ET IMPRESSION DU MANUEL

      Lors du setup de FlukeView, le manuel est installé sur votre
      PC dans un format électronique (.PDF). Afin de permettre la
      lecture et l'impression du manuel, il faut que le programme
      Acrobat Reader V7.0 (ou plus récent) soit installé sur votre
      système. Pour votre convenance, le logiciel nécessaire à
      installer Acrobat Reader se trouve sur le CD-ROM dans
      le répertoire \ACROBAT.
   ________________________________________________________________

   4. INSTALLATION SUR WINDOWS XP/VISTA/7 

      Il se peut que l'installation sur Windows exige l'autorisation
      d'agir comme administrateur de système afin de pouvoir
      réussir la mise au point de votre système. Setup est
      alors terminé et vous êtes invité à entrer en session comme
      administrateur et d'exécuter de nouveau Setup.exe.
      Pendant l'installation, le contrôle des utilisateurs Windows 
      pourrait vous envoyer un message à propos d'un 
      programme 'Setup.exe' voulant acceder à votre ordinateur.
      Dans ce cas, selectionnez continuer pour poursuivre 
      l'installation de FlukeView

      Pour utiliser l'aide en ligne sous Windows, voir la 
      base de connaissance Microsoft KB91777607 en installant 
      Winhlp32.exe et activer les macros

   6. PROBLEMES RENCONTRES AVEC LE FLUKEVIEW

      Veuillez noter les problémes suivants que vous pouvez
      rencontrer lorsque vous utilisez FlukeView ScopeMeter
      pour Windows:

      a. Si le redimensionnement d'une fenêtre entraîne de grands
         caractères de texte, désactivez le réglage Windows "True
         Type Fonts Only".

         Sélectionnez:  Démarrer, Paramètres, Panneau de
         configuration (Control Panel), Polices (Fonts). Dans le menu
         choisissez: Affichage, Options (View-Options) puis l'onglet
         True Type (True Type Fonts). Déselectionnez: Affichez les
         polices true type dans les programmes de mon ordinateur
         (Show only True Type Fonts in the programs on my computer).

      b. Si l'option d'affichage de Windows "Show window contents
         while dragging" [Afficher le contenu de la fenêtre en
         faisant glisser]  est validée, FlukeView affichera
         uniquement les mises á jour intermédiares tandis que l'on
         fait glisser les fenêtres d'écran de l'appareil.

      c. suite à l'insertion de modification de dernière minute dans
         le logiciel, des écart avec le fichier d'aide peuvent 
         apparaitre.
