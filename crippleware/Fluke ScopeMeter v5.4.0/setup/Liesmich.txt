   1. DEMO VERSION, AKTIVIERUNG UND UPDATES

      Diese FlukeView Version ist eine Demo Version mit 
      eingeschränkter Funktionalität. Um volle Funktionalität zu 
      erhalten muss der Aktivierungscode, welcher beim Kauf der 
      Software geliefert wird, im Aktivierungsfenster eingegeben 
      werden.
      Um FlukeView mit eingeschränkter Funktionalität zu benutzen 
      klicken Sie DEMO im Aktivierungsfenster.
      Diese Version ermöglicht zudem den Upgrade für Benutzern von
      FlukeView Version V4.x auf die Vollversion V5.x.

   3. LESEN UND DRUCKEN DES HANDBUCHS

      Während der Installation von FlukeView wird das Handbuch in
      elektronischem Format (.PDF) auf Ihrem PC installiert. Zum
      Lesen und Drucken des Handbuchs muß Acrobat Reader V7.0 (oder
      höher) auf Ihrem System installiert sein. Der Bequemlichkeit
      halber finden Sie die Installationssoftware für Acrobat
      Reader im Verzeichnis \ACROBAT Ihres CD-ROM.
   ________________________________________________________________

   4. INSTALLATION UNTER WINDOWS XP/Vista/7

      Bei der Installation unter Windows könnte es erforderlich
      sein, daß Sie Administrator-Rechte haben, um Ihr System
      erfolgreich aktualisieren zu können. Wenn dies der
      Fall ist und Sie haben nicht solche Rechte, wird das Setup-
      Programm abgebrochen, und Sie werden aufgefordert, sich
      nachträglich als Administrator anzumelden und Setup.exe
      erneut zu starten.
      Windows Vista verfügt über erweiterte Sicherheitsfunktionen,
      daher kann es vorkommen, dass während der Installation die 
      Windows Account Control eine Meldung ausgibt welche 
      besagt, dass ein unidentifiziertes Programm "Setup.exe" 
      Zugriff auf Ihren Computer nehmen möchte. In diesem Fall 
      klicken Sie "Zugriff erlauben" denn es ist das FlukeView 
      Setup Programm.

      Um die Hilfedatei unter Windows benützen zu können
      lesen Sie bitte den Microsoft Knowledgebase Artikel 
      KB91777607 betreffend der Installation von Winhlp32.exe
      und der aktivierung entsprechender Makros.

   6. BEKANNTE PROBLEME BEI FLUKEVIEW-ANWENDUNGEN

      Bitte beachten Sie folgende Probleme, auf die Sie bei
      Anwendung der FlukeView-ScopeMeter-Software für Windows
      stoßen könnten:

      a. Wenn die Größenänderung eines Fensters zu großen
         Text-Zeichnen führt, sollten Sie die Windows-Einstellung
         "True Type Fonts Only" unwirksam machen.

         Rufen Sie im Windows-Menü "Start", "Einstellungen" und
         "Systemsteuerung" (Control Panel) auf, doppelklicken auf
         "Schriftarten" (Fonts), danach wählen Sie "Ansicht",
         "Optionen..." (View-Options) und "TrueType", dort
         deaktivieren Sie die Option "Nur Truetype-Schriftarten
         anzeigen" (Show only True Type Fonts in the programs on my
         computer).

      b. Wenn die Windows Anzeige-Option "Show window contents while
         dragging" [Fensterinhalt beim Ziehen der Maus anzeigen]
         wirksam gemacht wird, zeigt FlukeView beim Ziehen der
         Instrumenten-Schirmfenster lediglich die zwischenzeitlichen
         Aktualisierungen.

      c. Aufgrund von Softwareverbesserungen in letzter Minute vor
         dem Release können kleine Abweichungen in der Hilfedatei
         auftreten.
