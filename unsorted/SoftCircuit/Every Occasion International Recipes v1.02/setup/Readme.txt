Overview
--------
Contains a library of 310 recipes from 16 countries--from Abalone to Zabione. Convenient search feature locates specific recipes, or lists all those recipes with a particular ingredient. You can add your own countries and recipes and there is even a feature to enter the names of friends and business associates and a list of their favorite recipes. Print recipes to any Windows-compatible printer.

Installation
------------
Before you can use this software, you must install it by running the setup program (SETUP.EXE).
